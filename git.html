GIT

Git est un logiciel de versionnement de code qui fonctionne avec un système d'arborescence à partir d'une branche master. Git fait constamment le lien entre un repository local et un repository hébergé par GitHub. Un repository est un dossier qui est configuré pour travailler avec git.

Configuration

Pour connaître la version du logiciel git : 

git --version

Pour configurer les données utilisateur de git :

git config --global user.name "monNom"
git config --global user-email "monEmail"

Il faut créer un dossier local qui va héberger les différents repositories, par exemple Git_Directory. Puis se placer dans Git_Directory et initialiser le dossier :

git init

Il faut ensuite importer un repository du master online :

git clone https://github/monAdresse

Principales instructions

Pour ajouter un fichier local au repository online : 

git add monFichier

Pour ajouter rapidement au repository online sans distinction de fichier : 

git add .

Pour synchroniser le repository online master avec le repository local origin :

git add .
git commit -m "monCommentaire"
git push origin master

Pour connaître l'état des modifications :

git status

Pour récupérer un fichier de travail :

git pull monFichier master
